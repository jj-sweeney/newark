# newark
For analyses of Newar Tui Chub _Siphateles newarkensis_

## Files
100-alignment.Rmd - description of alignment and sequencing success
100.1-do-align.sh - script for aligning to reference genome with BWA. Uses SAMTools and produces some alignment metrics and sorted and filtered .bam files

## Git Files
.gitignore - comments out subdirs outputs genome data bams and any *html file

## Directories
bams - for bams if desired 
data - for data
genome - for reference genome and indices
metadata - for metadata 
outputs - for intermediate outputs, e.g. outputs/100/ for outputs for 100-alignment.RMD
